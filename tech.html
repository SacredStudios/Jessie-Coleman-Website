A* Algorithm Using Arduino



Over the summer of 2022, I familiarized myself with Arduino, and by August, I designed a working simulation of the A* Pathfinding algorithm using an Arduino Mega Arduino and a 3.5 inch TFT screen. The algorithm supports the Manhattan heuristic. I used my 3D printer to create the case.
